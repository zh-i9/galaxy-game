A easy-to-play game from Gemini 3 pro.

## 功能使用说明
1. 暂停与退出:
- 战斗中，点击左上角的 || 按钮可暂停游戏。
- 在暂停菜单中，你可以选择 RESUME (继续) 或 EXIT GAME (退出到主界面)。
- 高分记录：游戏利用 localStorage 自动保存。你即使关闭浏览器，下次打开也能看到“BEST SCORE”。
2. 敌人 (Enemy Types):
- 🔺 红色三角：普通敌人，只会撞你。
- ⚡ 黄色小三角：极速敌人，速度很快。
- 🟪 紫色方块 (坦克)：血量超厚，移动缓慢，奖励高。
- ⚪ 白色菱形 (射手)：它不会主动撞你，而是会远程发射红色子弹瞄准你，请注意躲避子弹！
3. 掉落系统 (Loot):
敌人死后有概率掉落发光的小球，不需要点击，驾驶飞机碰触即可吃到：
- 🟢 绿色 (HP+)：直接回血。
- 🔴 红色 (DMG UP)：获得 5 秒钟的双倍伤害 Buff，期间你的子弹会变成红色。
- 🟡 金色 (COINS)：直接获得大量金币。
4. 难度成长:
- wave (波次) 越高，敌人生成速度越快，血量越厚。
- 商店依然存在，打完一波后进入商店升级，跟上敌人的强度。
